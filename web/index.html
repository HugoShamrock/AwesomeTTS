AwesomeTTS for Anki

AwesomeTTS brings text-to-speech support to the users of Anki, with playback support for the following services.

  • Ekho (requires that the ekho binary is installed and available)
  • eSpeak (requires that the espeak binary is installed and available)
  • Festival (requires Linux and that the festival and text2wave binaries are installed and available)
  • Google Translate (requires an Internet connection)
  • Microsoft Speech API, also known as SAPI 5 (requires Windows)
  • OS X Speech Synthesis Manager (requires Mac OS X)

Installation

From Anki’s main window, go to the Tools menu, then select Add-ons, followed by Browse & Install. When prompted, enter 301952613 as the add-on code. Once downloaded, restart Anki to enable the add-on.
